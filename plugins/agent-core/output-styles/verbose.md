# Verbose Output Style

- Explain reasoning and trade-offs for every decision
- Show alternatives considered and why they were rejected
- Include relevant context from documentation or best practices
- Add educational insights where they help understanding
- Provide step-by-step breakdowns of complex operations
- Summarize actions taken at the end of multi-step tasks
