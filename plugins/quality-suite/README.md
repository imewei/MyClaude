# Quality & Maintenance Suite

Consolidated suite for code quality, test automation, legacy modernization, and debugging. Optimized for Claude Code v2.1.12 with parallel agent execution and MCP server integration.

## üöÄ Features

- **Intelligent Debugging**: Systematic root cause analysis and AI-assisted debugging across any stack.
- **Test Automation**: Automated generation and iterative execution of test suites until zero failures.
- **Code Quality**: Multi-perspective reviews covering architecture, security, and best practices.
- **Technical Debt Management**: Identification, prioritization, and remediation of tech debt using ROI-based roadmaps.
- **Documentation Engineering**: Automated generation and synchronization of technical documentation and tutorials.
- **Refactoring & Cleanup**: Automated code improvements following SOLID principles and modern patterns.
